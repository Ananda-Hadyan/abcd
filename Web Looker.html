Looker

Di zaman sekarang ini mencari pekerjaan sangat susah sekali.Apalagi untuk para freshgraduate yang kebanyakan hanya bermodal ijazah lulusan S-1 Karena semakin berkembangnya zaman kebutuhan di dunia industri berkembang pesat.Dan juga keterbelakangan mengenai informasi lowongan pekerjaan juga menjadi masalah yang besar bagi para freshgraduate untuk mendapatkan pekerjaan.

Oleh karena itu dengan melihat kondisi ini yang dimana para freshgraduate kesusahan dalam mencari pekerjaan.Saya membuat sebuah web bernama Looker yang akan membantu para freshgraduate ini dalam mencari pekerjaan.di dalam web ini akan berisi tentang.

  • Berita tentang lowongan pekerjaan
  • Webinar
  • Pengembangan Soft skill dan Hard skill
  • Pelatihan membuka lapangan pekerjaan

Di harapkan dengan adannya web ini para freshgraduate dapat lebih mudah dalam mencari pekerjaan selain itu para freshgraduate juga akan di bekali dengan pengembangan skill agar tidak hanya bermodal ijazah saja tapi juga mempunyai kemampuan khusus yang mumpuni dalam pekerjaan nya
